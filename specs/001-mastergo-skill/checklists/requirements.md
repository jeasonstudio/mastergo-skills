| # | 问题 | 答案 |
|---|------|------|
| 1 | Scripts 的角色定位 | 完全替代 MCP，直接调用 MasterGo API |

- **架构变更**: 从依赖 MCP Server 改为自有 Node.js scripts 直接调用 API
- **环境变量**: `MASTERGO_TOKEN` (必需)、`MASTERGO_API_URL` (可选)
- **范围调整**: P3 (组件开发工作流) 延后到 v1.1
- **v1.0 范围**: 5 个 scripts（get-dsl、get-component-link、get-meta、parse-mastergo-url、extract-component-links）
